## Quick orientation

Minas Lórien is a private inspiration vault — dark, warm parchment surfaces (`#1a1a18` / `#1f1f1c`), one brass accent (`#a58a5c`), and a Newsreader-serif body paired with uppercase Raleway-light chrome at wide letter-spacing. Four category accents (Work blue, Creativity violet, Family moss, Inspiration brass).

- Tokens & semantic classes → `colors_and_type.css`
- Brand mark → `assets/tower-logo.svg` (always paired with the wordmark)
- Reference UI → `ui_kits/web-app/`
- Voice & visual rules → `README.md`
